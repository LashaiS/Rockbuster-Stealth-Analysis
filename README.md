# Rockbuster Stealth Analysis
Rockbuster Stealth:  SQL Analysis &amp; Data Visualization

## Introduction

Rockbuster Stealth LLC is a movie rental company that used to have stores around the
world. Facing stiff competition from streaming services such as Netflix and Amazon Prime,
the Rockbuster Stealth management team is planning to use its existing movie licenses to
launch an online video rental service to stay competitive. The task is to load all of Rockbuster’s data into a relational database management system (RDBMS). 

## Key Questions
● Which movies contributed the most/least to revenue gain?

● What was the average rental duration for all videos?

● Which countries are Rockbuster customers based in?

● Where are customers with a high lifetime value based?

● Do sales figures vary between geographic regions?

## Tools Used 
● PostgreSQL - used to store and query the Rockerbuster data

● SQL(pgAdmin/SQL Shell) -used to write queries, join tables, filter data, and calculate metrics

● Tableau - used to create visualizations and present insights to business managers

● Excel - used for reviewing exported query results


## Data Overview
The Rockbuster dataset represents a movie rental business and contains information about films, customers, payments, and rental activity. The data is structured in multiple tables 

### Key Tables Used 
● film - details about movies (title, genre, rental rate, and length)

● inventory - links film to physical inventory times 

● rental - records each rental transaction

● payment - shows how much customers paid and when

● customer - customer demographic and location information

● category - used to group movies by genre
